# Rays LLM Dashboard Frontend

React-based dashboard for the multi-model LLM system with real-time metrics and feedback interface.

## Setup

```bash
cd frontend
npm install
npm run dev
```

Dashboard will be available at http://localhost:3000

## Features

- **Chat Interface**: Interact with multiple LLM models simultaneously
- **Model Selector**: Choose which models to query
- **Metrics Dashboard**: Real-time performance metrics and model statistics
- **Feedback System**: Rate responses to help the system improve
- **Ensemble Responses**: Get aggregated responses from multiple models

## Building

```bash
npm run build
npm run preview
```
